### Voice-to-Text Web App

A responsive browser-based speech recognition app built with the Web Speech API.
Supports continuous dictation, live transcription, and clipboard export.

### Tech stack

- HTML5
- CSS3
- JavaScript (ES6+)
- Web Speech API
